### Python demo code for "Guided Image Filtering" (ECCV 2010)

by Ruby Childs, based on the Matlab code of Kaiming He (kahe@microsoft.com)

This code is for academic purpose only. Not for commercial/industrial activities.

#### Usage:

guidedfilter.py - guided filter implementation (Eqn(5), (6), (8) in the paper)
guidedfilter_color.py - guided filter for color guidance (Eqn(14), (15), (16) in the paper)

Run the four examples to see the results shown in the paper.
